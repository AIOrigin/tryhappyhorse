1. tracking hot trends,
2. turning trends into searchable template pages,

6. reviewing the data and expanding what works.

`SEO page = template page = conversion page`

## 8. Auto SEO SOP

### 8.1 Signal Capture SOP

Check these sources every day:

1. Google Trends
2. Google Suggest
3. Google News
4. Search Console rising queries
5. X trending posts
6. Reddit mentions
7. product leaderboards and benchmark sites
8. competitor landing pages and new indexed pages

For every new topic, capture:

1. trend name,
2. first-seen date,
3. source URL,
4. likely search intent,
5. commercial relevance,
6. whether it can be templated.

### 8.2 Topic Scoring SOP

Score each trend on a 100-point scale.

Scoring dimensions:

1. relevance to your audience: `30`
2. search intent strength: `25`
3. templating ability: `20`
4. speed window: `15`
5. monetization fit: `10`

Execution rules:

1. `80+`: publish within 24 hours
2. `60-79`: add to this week
3. `<60`: monitor only

### 8.3 Page Production SOP

For each selected topic:

1. define primary keyword,
2. define page intent,
3. choose page type,
4. generate title candidates,
5. generate page brief,
6. draft page,
7. add FAQ and related links,
8. add CTA,
9. add disclaimer,
10. publish and index.

### 8.4 QA SOP

Before publishing, confirm:

1. not pretending to be official,
2. title matches search intent,
3. intro answers the query immediately,
4. CTA is visible above the fold,
5. page is different from other pages,
6. internal links are added,
7. schema is valid,
8. disclaimer exists,
9. page has one clear next step,
10. no fluff-only AI writing.

### 8.5 Publish SOP

On publish:

1. add to sitemap,
2. link from homepage or relevant hub,
3. link from at least 2 related pages,
4. submit to Search Console,
5. post to distribution channels,
6. track first index time,
7. track impressions within 7 days.

### 8.6 Distribution SOP

Do not wait for search only.

Each important page should have:

1. one X post,
2. one short social post,
3. one creator-facing prompt sample,
4. one email or newsletter mention,
5. one forum or community post if relevant.

### 8.7 Review SOP

Review every page at:

1. Day 3,
2. Day 7,
3. Day 14,
4. Day 30.

Decision logic:

1. no index: fix title, links, sitemap, uniqueness
2. indexed but no clicks: fix title and description
3. clicks but no conversion: fix CTA and landing flow
4. strong page: expand variants and comparisons

## 13. What Not to Do

Do not do the following:

1. pretend to be official,
2. copy news article wording without adding structure or value,
3. build only one page and stop,
4. publish trend pages without CTA,
5. publish generic AI fluff,
6. ignore comparison and alternatives intent,
7. wait for perfect data before shipping,
8. overbuild multiple domains before the first one works.
